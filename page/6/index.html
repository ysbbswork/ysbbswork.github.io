机械键盘

发表于 2018-12-28 | | 阅读次数

一款好的机械键盘是敲代码，工作的动力源泉

G80-3082

初步体验了一下，很爽，我想这是每个新入机械键盘的人的共同感受吧。

茶轴的噪音也是够大的了，但是真的很爽，不知道青轴声音在公共区域实际会是什么样子。

总之即使是最入门级的cherry键盘，已经让我爽到了。

下一次自己的工作电脑 要更好的键盘才行

攒点钱可以可以 filco 或 极度酷冷 ikbc

谈几点G80-3082 MX2.0的缺点吧：

觉得功能F键和数字键间隔太小，很容易误触。

突然觉得小键盘数字键可有可无？很占地方，在抽屉式键盘，将键盘移动到中间，小键盘会抵住右边，很难受。

虽然我个人很习惯用右侧小键盘，但是这个习惯现在觉得可有可无了，右侧键盘区使用频率不高，并且占位比较大，对于桌面面积利用率和效率折中之后，我觉得下次我可以可以省去。

这款键盘我买的是白色的，ABS的键盘 塑料感非常强烈，也非常的复古，特别是这种白色灰刻字的外观，简直勾起小时候的好多回忆。

这款键盘还有个优点就是半高的键帽，比较浅，敲字也不会很累。（但这个特点也被很多人诟病，被称为平胸cherry）

是否是优点，我不能确定，但绝对可以说是一个特点。

黑猫白猫，我觉得自己用的顺手的就是好猫，机械键盘品类繁多，网上评测也超级多，但众说纷纭，每个人又无法跟每个键盘进行磨合，如果没有经历过，就没有资格去评价什么，所以一个适合自己的键盘不仅仅是价格和质量，还有使用习惯上的不断磨合，其实，我敲下这么多废话的时候，就是在和我的键盘在进一步的磨合，这个时候的键位紧凑好像需要再熟悉熟悉，还不熟悉导致误触的概率还是很大的。

无论怎么样，开心最好，这个键盘能带来的开心远远超过它本身的价格了，总之喜欢。

后记：

换了新实验室，位置比楼上小，为了节省空间，只坐在右侧一边，所以键盘从抽屉移到桌面上，也习惯了，蛮舒服的。

买了套高键帽白红主题换上，平胸键盘一下子 变高大上了，越用越舒服，看来平胸键盘为人诟病是有原因的。

但是敲久了又发现一个个人问题，一直以来打字都是二指禅的方式，敲久了手指会痛，不能敲太快，很难受。

尝试用正确的打字法打字争取用上所有的手指，但是不使用二指禅几乎快忘记其它按键位置，敲下这一行，快按了十几下退格了，累死。

写作是多么重要却一直被忽略的事情

发表于 2018-11-30 | | 阅读次数

输出是多么重要的东西，却一直被忽略。

知乎是一个来进行输入远远大于输出的一个社区。这里的输出方式很简单，也是最有效的一种方式，写作。

知乎上开了很多问答：例如如何赚的一百万，如何财富自由，如果利用副业赚钱，等等。

下面点赞很高的回答中都会有很多个推荐你去进行自媒体创作，也就是通常来说的 写作

但写作不仅仅可以作为一种给你赚钱的技能方式，更多的是一种学习方式，因为输入决定输出，当你开始尝试写作的时候，你就被迫的要去大量的吸收知识，并且要消化知识，并由自己整理转述出来。

当大部分人决定要写点什么的时候才发现，自己不仅仅是缺乏调理的语言组织能力，讲故事的能力，更多的时候会发现，自己没什么可以写的，自己想写的那个东西，几句话可能就说完了，无法旁征博引，做不到引经据典，更多的时候像是偏向于经验之谈，夸夸之谈，如果有人来考究，或者杠一下自己，就会乱了分寸，甚至开始自我怀疑起来。

套接字设置非阻塞，可以对套件字设置非阻塞，这样就可以不必为等待一个套接字而整个程序（进程，线程）阻塞在那里，

设置了套接字不阻塞，则如果此套接字队数据还不可用，则任何阻塞函数调用之都会失败，并且返回一个EWOULDBLOCK错误，每次出错了要判断一下是不是阻塞错误，如果是的话就忽略它，但是这种效率低下，

更好的办法是使用select()函数立即轮询许多套接字集，检测其中套接字是否有活动。

或者还有一种方法，就是使用多线程，每个线程接管一个连接connection，阻塞也是阻塞那个管理connection的线程，并不影响其它线程。
